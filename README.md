# Modern Todo App

A premium, feature-rich Todo application built with React, Tailwind CSS, and Framer Motion. Designed for a beautiful and fluid user experience.

## ✨ Features

- **Modern UI/UX**: Glassmorphism effects, smooth animations, and a clean interface.
- **Light & Dark Mode**: Seamlessly switch between themes with persistence.
- **Task Management**:
  - Add, Edit, Delete, and Toggle tasks.
  - Set **Priorities** (Low, Medium, High).
  - Categorize tasks (Personal, Work, Shopping, Fitness).
  - Set **Due Dates**.
- **Smart Filtering**: Filter by All, Active, or Completed status.
- **Search**: Real-time search functionality.
- **Progress Tracking**: Visual statistics and progress bar.
- **Gamification**: Confetti celebration when completing all tasks!

## 🚀 Getting Started

### Prerequisites

- Node.js installed on your machine.

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/yourusername/todo-list.git
    cd todo-list
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

## 🛠️ Tech Stack

- **React**: UI Library
- **Tailwind CSS**: Styling & Design System
- **Framer Motion**: Animations
- **Context API**: State Management
- **Canvas Confetti**: Fun effects
- **React Icons**: Iconography
- **Vite**: Build tool

## 📁 Project Structure

```
src/
├── components/     # UI Components (TodoItem, Navbar, etc.)
├── context/        # React Context (TodoContext, ThemeContext)
├── hooks/          # Custom Hooks (useTodos, useTheme)
├── App.jsx         # Main Application Entry
├── index.css       # Global Styles & Tailwind Configuration
└── main.jsx        # Root Render
```

## 🎨 Customization

You can customize the color palette in `tailwind.config.js`.
